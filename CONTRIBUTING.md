# Contributing

Thank you for your interest in contributing!

## AI-Assisted Contributions

Contributors may use AI coding assistants (ChatGPT, Copilot, Cursor, etc.).

**Requirements:**

- Disclose AI use in your PR description or commit message
- Be able to explain, debug, and maintain your code
- You (not the AI) are responsible for correctness, security, and license
  compliance

**Review:** All contributions are reviewed to the same standards regardless
of how they were created.

## Security

This project uses [Red Hat Product Security SKILLS](https://github.com/RedHatProductSecurity/prodsec-skills)
to automatically review code for security issues. PRs that introduce
vulnerabilities will be flagged and must be resolved before merging.
